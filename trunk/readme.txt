== Description ==

Some hosts are not optimized at the level of PHP, so the same site may use different amounts of RAM. This plug is suitable for the rapid test used and allocated memory.

Just activate the plugin. As a result, in the upper right corner there is the value of memory used in relation to dedicated hosting.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload folder `hiweb-core` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'hiWeb Memory Usage' and menu in WordPress
3. Just activate the plugin. As a result, in the upper right corner there is the value of memory used in relation to dedicated hosting.
